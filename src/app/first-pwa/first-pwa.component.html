How can you create your First PWA?

  1. Run ng add @angular/pwa
  2. ngsw-config.json, manifest.webmanifest files will be created.
  3. Setup the ServiceWorker cache config on angular.json
  4. Add the ServiceWorker provider on main.ts
  5. Build and test the PWA in production mode.
